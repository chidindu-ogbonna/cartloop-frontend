## Setup

Clone the Frontend Repo

`git clone https://github.com/chidindu-ogbonna/cartloop-frontend`

Install the required packages

`cd cartloop-frontend`

`yarn install`

Start application

`yarn start`

To Deploy

`yarn deploy`
